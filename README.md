# Tesheng Phone - Landing Page

一個以高端智慧型手機為主題的產品形象頁面。

## 專案背景

本專案根據以下目標用戶輪廓（User Persona）進行設計：

- 目標用戶：30 多歲的職業女性，擁有碩士學歷並管理大型團隊
- 生活狀態：居住於桃園，行程繁忙，從未購買過類似產品
- 設計目標：以簡潔、高端的視覺風格傳遞產品價值，
  並透過清楚的 CTA 引導用戶採取行動

## Demo

[點此觀看](https://tesheng-phone.netlify.app/)

## 使用技術

- HTML5 語義化標籤
- CSS3 / Sass
- BEM 命名方法論
- RWD 響應式設計

## 專案特色

- 純 CSS 實作漢堡選單開關動畫
- 純 CSS Popup 表單互動
- 針對不同裝置尺寸進行響應式設計

## 本地端執行

git clone repo連結
cd 專案資料夾
